Integrate AI-powered CV screening into your own platform. Send CVs to our API and get instant ranked results with match scores, skill gaps, interview questions and hiring recommendations.

✓ REST API GPT-4o powered JSON responses Base URL: https://topcandidate.pro/api/v1

Introduction

The TopCandidate API lets you screen CVs programmatically using our AI pipeline. You can integrate it into your ATS, HR platform, job board, or any custom workflow.

Our 3-step AI pipeline extracts CV data, parses job requirements, then scores candidates across 6 dimensions — running twice and averaging for consistency.

# Quick example — screen a CV in one API call curl -X POST https://topcandidate.pro/api/v1/screen \ -H "X-API-Key: tc_live_your_key_here" \ -F "cv_file=@candidate.pdf" \ -F "job_description=We are looking for a Data Analyst..." \ -F "job_title=Data Analyst"

Authentication

All API requests must include your API key in the X-API-Key header.

🔑

Generate your API key in the dashboard under Settings → API Keys. Your key starts with tc_live_

⚠ Never share your API key or commit it to version control. Revoke and regenerate immediately if compromised.

Scoring dimensions

Each CV is scored across 6 dimensions, each out of 20 (total 100):

DimensionWhat it measures
Technical SkillsMatch of listed skills vs JD requirements
Experience MatchYears and type of experience relevance
EducationDegree relevance and qualification match
Achievement QualityMeasurable results and impact shown
Role ProgressionCareer growth and trajectory
Keyword RelevanceJD keyword coverage across the CV

Quickstart — full workflow

Here's a complete example: screen 3 CVs for a role and get ranked results.

import requests import os API_KEY = os.environ["TOPCANDIDATE_API_KEY"] BASE = "https://topcandidate.pro/api/v1" HEADERS = {"X-API-Key": API_KEY} JD = """We are hiring a Data Analyst with 3+ years experience. Required: Python, SQL, Power BI, Excel. Strong communication skills. Must have experience with large datasets and reporting.""" cv_files = ["candidate1.pdf", "candidate2.pdf", "candidate3.pdf"] results = [] for cv_path in cv_files: with open(cv_path, "rb") as f: res = requests.post( f"{BASE}/screen", headers=HEADERS, files={"cv_file": (cv_path, f, "application/pdf")}, data={"job_description": JD, "job_title": "Data Analyst"} ) results.append(res.json()) # Sort by score ranked = sorted(results, key=lambda x: x["overall_score"], reverse=True) for i, r in enumerate(ranked, 1): print(f"\n#{i} {r['candidate_name']}") print(f" Score: {r['overall_score']}/100 | {r['recommendation']}") print(f" Skills: {r['skills_coverage_pct']}% match") print(f" Report: {r['report_url']}")
